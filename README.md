# Full-Stack Task Management App

**Task Manager*** is a complete full-stack web application demonstrating modern web development skills. Built with Python Flask backend, SQLite database persistence, and responsive Bootstrap frontend.

# Features
- **Add Tasks** with title, description, due date
- **Toggle Complete** (todo ↔ done with visual feedback)
- **Delete Tasks** with confirmation
- **Persistent Storage** - data survives refresh (SQLite)
- **Responsive Design** - mobile + desktop
- **Production Deployed** - Render.com live URL
